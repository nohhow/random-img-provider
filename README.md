# random-img-provider
resources/static/images 하위 폴더를 기준으로 End-Point를 제공하고
각 End-Point 요청에 따라 Random한 이미지를 제공하는 API

- 개발 : spring boot
- 배포 : docker + fly.io


---

현재는 귀여운 "망고"(요크셔테리어) 사진을 제공하는 것에 한정되어있음.
요청 주소 : https://ran-img-provider.fly.dev/random-image
